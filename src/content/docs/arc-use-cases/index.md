Here we take a closer look at some typical scenarios that scientists might face. With these examples we aim to provide best practices for ARC-supported data annotation allowing to generate machine-readable, interoperable and reproducible data and metadata.

## Request or contribute a use-case

1. Check-out the [ARC Annotation Use-Cases project board](https://github.com/orgs/nfdi4plants/projects/14) for open use-cases
2. If your use-cases is not yet raised, please request it via [issue](https://github.com/nfdi4plants/nfdi4plants.knowledgebase/issues)
3. Once you want to contribute your own use-case, please open a pull request referencing the related issue.

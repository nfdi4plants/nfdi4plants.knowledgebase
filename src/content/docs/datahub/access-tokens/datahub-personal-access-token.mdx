:::note
This guide shows you how to generate a **Personal Access Token (PAT)**.  
A PAT can grant access to **all ARCs** of one user.
:::

<Steps>

1. Sign in to the [DataHUB](https://git.nfdi4plants.org/)
2. Navigate to your [user settings](/nfdi4plants.knowledgebase/datahub/navigation-settings/datahub-user-settings)
2. Open the `Access Tokens` settings in the sidebar `(1)`
3. Enter or adapt the required token information `(2)` (more about token information [here](/nfdi4plants.knowledgebase/datahub/access-tokens#token-information))
4. Click <btn>Create personal access token</btn> `(3)`

    ![](@images/datahub/access-tokens/datahub-access-token-personal-01.drawio.png)

5. `Your new personal access token` appears on top. Copy `(4)` and use it in the desired application.

    ![](@images/datahub/access-tokens/datahub-access-token-personal-02.drawio.png)

</Steps>

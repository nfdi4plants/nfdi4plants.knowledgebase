Ok now I uploaded my protocol, how do I create the metadata about the process and important information in it?

To add a step of the experimental process as an annotation, mark the word and right-click to open the BOAT context menu.

Inside the upper context menu, the marked term can be placed inside a new annotation as a **Key, Term or Value**. In the lower part, the selected word can be added to the annotation lastly created as the same types.
Here the Key refers to the header of the annotation and Term with an optional Value is the content.

<ViolaSays>
Got it! So the Key serves as the header which describes the step of the process. The Term is the actual content which can be unit or a substance for example. The Value is an optional number.
</ViolaSays>

After clicking on an action inside the context menu, an annotation note is created at the edge of the document with the marked word in the respective text field.
They can be expanded or downsized again, leading to a small speech bubble icon.

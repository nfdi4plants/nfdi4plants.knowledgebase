Added enough annotations? Then just click on the download button and decide if the output metadata is converted into a **.xlsx** or **.json** file.

    <ViolaSays>
    After some time I selected enough metadata from the process in my protocol and want to get my metadata file. Using the download button lets
    me select two different file format. Which one do I choose?
    </ViolaSays>

It depends on which file format fits to your workflow or environment the best, for example XLSX works naturally great with MS Excel or the [ARCitect](https://nfdi4plants.github.io/nfdi4plants.knowledgebase/arcitect/). The JSON format is more suitable for developing environments but can also be added to the ARC.

And that's it!



If you want to annotate more metadata in a new protocol, you can delete it using the red delete button or just upload a new file which overwrites the current document.
If the protocol inherits multiple processes, you can just delete all your added annotation after you downloaded them.

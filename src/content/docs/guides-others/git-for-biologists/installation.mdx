<ViolaSays>

If Git is the infrastructure for our protocol knowledge base, how do we install it?

</ViolaSays>

Git enables structured tracking of text files such as Markdown SOPs.


<Card title="Install Git" icon="download">

Download Git from:

https://git-scm.com/download/

Follow the installation instructions for your operating system.

</Card>


<Card title="Initial Configuration" icon="person">

<Steps>

1. Open a terminal.
2. Configure your identity:

    ```bash
    git config --global user.name "Your Name"
    git config --global user.email "your.email@example.com"
    ````

</Steps>

</Card>

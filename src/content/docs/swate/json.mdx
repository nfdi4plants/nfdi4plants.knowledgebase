Utilizing [ARCtrl](https://github.com/nfdi4plants/ARCtrl), Swate supports multiple different import and export formats for JSON. 

## Export

The "File Export" function can be found in the <a href={SwateOverview.src} target="_blank">Sidebar</a>.

<Tabs syncKey="swateHost">
<TabItem label="Browser">

You can export your metadata sheet simply by clicking on "Download".

</TabItem>
<TabItem label="Excel">

In contrast to the web version of Swate, you cannot simply download your annotation tables.

In order to do so, you have to create a [top level metadata sheet](/nfdi4plants.knowledgebase/swate/top-level-metadata).


</TabItem>
</Tabs>

![](@images/swate/json-export.png)



## Import 

The "File Import" function can be found next to the export function in the side bar. For importing a filled Swate table, select the same format that was used for the export, upload your file and select your desired import options.

![](@images/swate/json-import.png)

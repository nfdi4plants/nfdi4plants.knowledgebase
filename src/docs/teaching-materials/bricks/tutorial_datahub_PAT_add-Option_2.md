2. Clone your project via

```bash
git clone https://oauth2:<YourPAT>@git.nfdi4plants.org/<YourUserName>/<YourProjectName>
```

> Replace `<YourPAT>`, `<YourUserName>`, `<YourProjectName>` with your personal information.
